◆ MAX_BIGINT_GROUPS

#define MAX_BIGINT_GROUPS   256

The maximum number of groups of unsigned longs in a bigint.

The max value of any bigint is then equal to (1 << (MAX_BIGINT_GROUPS << 5)).
