Detailed Description

Module typedef struct.

This struct consists of: I. usermode: Toggles usermode II. enabled: Toggles if the module is enabled III. initialized: Toggles if the module has been initialized IV. name: The name of the module. Similar to a import in Java, but not backwards V. description: The description of the module. This description is displayed with the man function VI. dependencies: List of strings of dependencies. Each dependency's name should follow the convention established by name.
