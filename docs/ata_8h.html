void sect_read_atapio ( uint32_t  target_address,
uint32_t  LBA,
uint8_t  sector_count 
)

Reads a sector from the hard drive.

Reads sector_count*512 bytes from the hard drive in ATA PIO mode.

Parameters
target_addressThe target to modify when the function is finished. When the function is done running, target will have all 512 bytes in it.
LBAThe amount of sectors to offset the read by (0x0 = first sector, 0x1 = second sector, etc.)
sector_countThe amount of sectors in total to read. The sectors read are equal to {LBA<x<LBA+sector_count}
